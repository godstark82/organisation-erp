## Using Next.js starter kit
Kickstart your project with the [Next.js Starter Kit](https://github.com/intentui/next.js), pre-configured and ready to use:

```bash
npx create-next-app@latest -e https://github.com/intentui/next.js app
```




## Source code
To install Intent UI manually, clone the latest version from the [GitHub repository](https://github.com/intentui/next.js).
